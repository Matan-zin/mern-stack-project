#### Prerequisites:
    - docker
    - docker-comopse
    - node

#### Installation:

verify docker.socket are active
```
sudo systemctl status docker.socket
```
navigate to docker-compose directory 
```
cd docker-compose && sudo docker-comopose up --build
```
navigate to localhost on your browser
